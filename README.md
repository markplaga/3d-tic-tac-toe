# 3D Tic-Tac-Toe

A two-player 3×3×3 Tic-Tac-Toe game that runs entirely in the browser.

## Play online

https://markplaga.github.io/3d-tic-tac-toe/

## Play locally

Open `index.html` in a modern web browser.

## Deployment

The site is automatically deployed to GitHub Pages whenever changes are pushed to `main` using `.github/workflows/pages.yml`.
